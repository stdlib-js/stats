
{{alias}}( 位 )
    Returns the excess kurtosis of a Planck distribution with shape parameter
    `位`.

    If `位 <= 0`, the function returns `NaN`.

    Parameters
    ----------
    位: number
        Shape parameter.

    Returns
    -------
    out: number
        Excess kurtosis.

    Examples
    --------
    > var v = {{alias}}( 0.1 )
    ~6.0100
    > v = {{alias}}( 1.5 )
    ~8.7048
