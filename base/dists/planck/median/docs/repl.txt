
{{alias}}( 位 )
    Returns the median of a Planck distribution with shape parameter `位`.

    If `位 <= 0`, the function returns `NaN`.

    Parameters
    ----------
    位: number
        Shape parameter.

    Returns
    -------
    out: integer
        Median.

    Examples
    --------
    > var v = {{alias}}( 0.1 )
    6
    > v = {{alias}}( 1.5 )
    0
