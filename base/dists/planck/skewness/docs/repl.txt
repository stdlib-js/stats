
{{alias}}( 位 )
    Returns the skewness of a Planck distribution with shape parameter `位`.

    If `位 <= 0`, the function returns `NaN`.

    Parameters
    ----------
    位: number
        Shape parameter.

    Returns
    -------
    out: number
        Skewness.

    Examples
    --------
    > var v = {{alias}}( 0.1 )
    ~2.0025
    > v = {{alias}}( 1.5 )
    ~2.5894
