
{{alias}}( 位 )
    Returns the standard deviation of a Planck distribution with shape parameter
    `位`.

    If `位 <= 0`, the function returns `NaN`.

    Parameters
    ----------
    位: number
        Shape parameter.

    Returns
    -------
    out: number
        Standard deviation.

    Examples
    --------
    > var v = {{alias}}( 0.1 )
    ~9.9958
    > v = {{alias}}( 1.5 )
    ~0.6080
