
{{alias}}( 位 )
    Returns the expected value of a Planck distribution with shape parameter 位.

    If `位 <= 0`, the function returns `NaN`.

    Parameters
    ----------
    位: number
        Shape parameter.

    Returns
    -------
    out: number
        Expected value.

    Examples
    --------
    > var v = {{alias}}( 0.1 )
    ~9.5083
    > v = {{alias}}( 0.5 )
    ~1.5415
