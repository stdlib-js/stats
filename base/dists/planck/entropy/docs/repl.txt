
{{alias}}( 位 )
    Returns the differential entropy of a Planck distribution with shape
    parameter `位` (in nats).

    If `位 <= 0`, the function returns `NaN`.

    Parameters
    ----------
    位: number
        Shape parameter.

    Returns
    -------
    out: number
        Entropy.

    Examples
    --------
    > var v = {{alias}}( 0.1 )
    ~3.3030
    > v = {{alias}}( 1.5 )
    ~0.6833
