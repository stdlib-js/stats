
{{alias}}( 位 )
    Returns the variance of a Planck distribution with shape parameter `位`.

    If `位 <= 0`, the function returns `NaN`.

    Parameters
    ----------
    位: number
        Shape parameter.

    Returns
    -------
    out: number
        Variance.

    Examples
    --------
    > var v = {{alias}}( 0.1 )
    ~99.9167
    > v = {{alias}}( 1.5 )
    ~0.3697
