
{{alias}}( x )
    Computes the arithmetic mean of an array using pairwise summation.

    If provided an empty array, the function returns `NaN`.

    Parameters
    ----------
    x: Array<number>|TypedArray
        Input array.

    Returns
    -------
    out: number
        The arithmetic mean.

    Examples
    --------
    > var x = [ 1.0, -2.0, 2.0 ];
    > {{alias}}( x )
    ~0.3333
