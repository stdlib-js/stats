
{{alias}}( x )
    Computes the range of an array.

    If provided an empty array, the function returns `NaN`.

    Parameters
    ----------
    x: Array<number>|TypedArray
        Input array.

    Returns
    -------
    out: number
        Range.

    Examples
    --------
    > var x = [ 1.0, -2.0, 2.0 ];
    > {{alias}}( x )
    4.0
