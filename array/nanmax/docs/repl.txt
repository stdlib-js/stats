
{{alias}}( x )
    Computes the maximum value of an array, ignoring `NaN` values.

    If provided an empty array, the function returns `NaN`.

    Parameters
    ----------
    x: Array<number>|TypedArray
        Input array.

    Returns
    -------
    out: number
        Maximum value.

    Examples
    --------
    > var x = [ 1.0, -2.0, NaN, 2.0 ];
    > {{alias}}( x )
    2.0
